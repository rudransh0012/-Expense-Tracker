# 💰 Expense Tracker

A web-based Expense Tracker application that allows users to manage their personal finances by recording income and expenses, categorizing transactions, and displaying financial summaries.

---

## 📌 Project Overview

This project is built using **HTML**, **CSS**, and **JavaScript**. It enables users to:
- Add income and expenses
- Categorize transactions (e.g., Food, Transport, Entertainment)
- View total income, total expenses, and net balance
- See all transactions in a list format
- Enjoy a responsive and clean UI

---

## 🚀 Features

- ✅ Add and delete transactions
- ✅ Categorize expenses
- ✅ Income vs Expense summary
- ✅ Responsive layout (Mobile, Tablet, Desktop)
- ✅ Local storage support (data saved on browser)
- ✅ Input validation and error messages

---

## 🛠️ Tech Stack

- **HTML** – Structure
- **CSS** – Styling
- **JavaScript** – Functionality
